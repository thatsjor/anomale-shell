I use EndeavourOS and these are my personal dots, made for me.
These dotfiles appear to work on a fresh installation of EndeavourOS, run just after logging into tty on a 'No Desktop' installation. 

You should, in theory, be able to run these on base arch linux, but make sure the extras repo is enabled, and install yay first. Make sure GPU drivers are taken care of.

My default mangowm bindings will be strange. You can look at the config file to see them, but the important ones to be able to navigate to the config once in mangowm are:

SUPER+Tab = Spawn Terminal
SUPER+q = exit tile
Alt+Space = Apps Menu
Super+Space = Shutdown/Power Menu
